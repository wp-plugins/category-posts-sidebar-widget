== Description ==

This plugin provides site admin to use this widget on sidebars. The widget has options to set the parameters for multiple categories selection, number of posts, comments count, sorting, widget class, title class, show date and featured image thumbnails.

== Installation ==

1. Upload `catagory-posts-sidebar-widget.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress.

After activating the plug-in, go to widget area and drag this widget and drop in any sidebar section, then set all parameters, save and you are all done.
